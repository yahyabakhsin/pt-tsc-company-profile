# Custom Hooks (`src/hooks/`)

This directory houses global, reusable React hooks.

## Recommended hooks:
- `use-media-query.ts` - Breakpoint checking.
- `use-mounted.ts` - Prevent hydration mismatch errors.
- `use-active-section.ts` - Intersection observer for scroll detection.
- `use-debounce.ts` - Rate-limit input search fields.
